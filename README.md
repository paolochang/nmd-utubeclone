## Utubeclone

Implemented reverse engineered YouTube application. NodeJS, Express and ReactJS are used.

### Setup

Install `node_modules` in server side

```bash
$ cd server
$ npm install
```

Install `node_modules` in client side

```bash
$ cd client
$ npm install
```

### Start

Run server

```bash
$ cd server
$ npm run dev
```

Run client

```bash
$ cd client
$ npm start
```
